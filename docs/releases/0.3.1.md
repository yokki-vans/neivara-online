# Neivara Online 0.3.1 — character art replacement

0.3.1 replaces the rejected procedural humanoid roster with a production-ready,
license-auditable character pipeline built around KayKit Adventurers 1.0 (CC0).
It does not claim that the current vertical slice is a complete commercial MMO.

## Character roster

- twenty runtime GLBs cover five races, two genders and two archetypes;
- male warriors use knight or barbarian silhouettes, male mages use the mage
  silhouette, female warriors use the rogue silhouette, and female mages combine
  the female rogue body with mage equipment on the shared skeleton;
- light and dark elves have skinned pointed ears, dwarves have shorter/wider
  proportions, and orcs have green skin, larger proportions and skinned tusks;
- every variant includes visible class equipment and six gameplay clips:
  `idle`, `run`, `attack`, `cast`, `hit`, and `death`;
- the creation screen loads and switches all twenty animated previews.

## Licensing and reproducibility

- the four unmodified source GLBs and CC0 license are vendored under
  `third_party/kaykit/adventurers`;
- source SHA-256 values, the license notice and runtime GLB statistics are checked
  by `npm run assets:check:3d`;
- `npm run assets:generate:characters` rebuilds the roster deterministically with
  Blender 4.5 LTS;
- no Lineage II models, textures, rigs, animations or extracted game files are
  used.

## Release evidence

- all 20 preview combinations loaded in browser QA without the model error state;
- 131 repository tests passed and 6 database-dependent tests were skipped without
  a configured PostgreSQL instance;
- ESLint, TypeScript, asset verification and the production build passed;
- every runtime character remains under the 1 MB browser budget, with the roster
  GLBs near 0.5 MB each after unused animations and equipment are removed.
